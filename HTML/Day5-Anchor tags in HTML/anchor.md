## Anchor tags in HTML

_Anchor tag in HTML is an inline element generally used to create Links which will be redirecting us to any other HTML pages or contents. It comes up with multiple attributes such as href which holds the actual link or path of the resources where we need to move by clicking the specified link._

_We can use target attribute to open the link in different location. It accepts values such as `_self` which is default value `_blank` which helps us to open the link in new tab and so on._
