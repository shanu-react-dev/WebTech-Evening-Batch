## Tables

_Tables in HTML is the representation of Rows and columns to show the data in Structured format. We can create a table in HTML using `<table></table>` tag. It includes multiple tags inside it such as thead which stores the heading data in the table, `<tbody>` which stores the body data in the table and `<tfoot>` tag which stores the summarized data of the table._

_To create a row in table we have `<tr>` tag and to add the data in the table we can use `<th>` for heading data and `<td>` for body data inside a row._

## Rowspan and Colspan

_Rowspan and Colspan is an attribute which helps to merge two or more column ceslls or row cells inside a table. It accepts the values or count of the cells needs to be merged._
