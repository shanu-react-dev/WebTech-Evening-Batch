## Web Technology
*Web Technology is a technology which is used to create websites or Web Apps. It contains Frontend, Backend and Databases.*

## Websites and WebPages
*Website is a collection of multiple web pages and webpage is a single HTML page or document. A webpage contains data such as text data, image, videos, audios etc.*

## How website works
*Whenever we search any website using their URL first it makes one HTTP or HTTPs request to the server then server will process the request to the databases and if the data is availabel then it will return the data in the form of files such as HTML, CSS and JS or sometimes in JSON or XML.*

## DNS
*DNS stands for Domain Name Server or System it is used to translate the URL into an IP address.*

## IP address
*IP stands for Internet Protocols which contains set of rules to get the data from the server. It is an address of another system or devices.*

## Browser
*Browser is a tool which is used to display or show the data in visual format. It consists of multiple features such as read aloud history etc.*

## Browser Engine
*Browser engine is a component generally responsible for Executing the code which we are getting from the servers.*
