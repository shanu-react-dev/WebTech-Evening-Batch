## HTML
*HTML stands for Hyper Text Markup Language. It is a standard markup language used to create the structure of a webpage. It was introduced by Tim Berners Lee in 1991.*

## HyperText
*A text which contains some extra information known as Hyper text. In case of HTML tags can be considered as Hyper Text because it contains the information how the content will appear in the browser.*

## Markup Language
_A language which is used to define the structure of a webpage known as markup language._

## Structure Code
```html
<!-- Declaration of HTML 5th version or current version -->
<!DOCTYPE html>
<!-- html is a root element -->
 <!-- lang is an attribute to set the language -->
<html lang="en">
    <!-- head tag includes the data about the webpages it contains metadata related to webpages. -->
    <head>
        <!-- charset is used to define the character set in the entire file -->
        <meta charset="UTF-8">
        <!-- viewport is a space for showing the content -->
        <meta name= "viewport" content="width=device-width initial-scale=1.0">
        <!-- title tag is used to give the name of the webpages. -->
        <title>Document</title>
    </head>
    <!-- Body tag contains the data which can be visible to our users. -->
    <body>

    </body>
</html>
```
## Tags
*Tags in HTL are keyword which is predefined. Anything written inside angular braces known as tags. It is of two types.*

- Paired Tags: The tag which is having opening tag as well as closing tag known as paired tags. e.g. `<h1></h1>`, `<p></p> etc...`
- Unpaired Tags: This is also known as self closing tag. we do not need to close this explicitly. it has only opening tags or closing tags. e.g. `<img>`, `<br>`, `<hr>` etc..
