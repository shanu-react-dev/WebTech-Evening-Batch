## Media Tags in HTML

_Media tags in HTML is used to embed the media files in the webpages or data related to media such as image video audio etc.._

- Image: To add image data we have `<img>` tag and it includes some attributes such as `src` and `alt` where src can hold the path of the images it can be relative (path of a file in the same directory) or absolute path (URL) alt stands for Alternate Text and it holds the data about the image in case the image path is not correct or delted then it will show the image description.

### syntax

```html
<img src="path" alt="value" />
```

- Audio: To add audio data we can use `<audio>` tag and it comes up with several attributes such as `src`, `controls`, `loop`, `autoplay`, `muted`. Each and every attributes are having different task such as src holds the path of the resource and controls add controls like play pause and mute button to control the audio, loop helps to play the audio in loop without any pause, muted will mute the audio by default.

- Video: To add video file in the webpage we can use `<video>` tag it also has several attributes such as `src`, `controls`, `loop`, `autoplay`, `muted` and each and every thing is having different task similar to audio tag.

- `<iframe>` tag: This tag helps to embed any other html resources such as maps or any youtube video in my webpage.
