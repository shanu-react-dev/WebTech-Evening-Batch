## Form In HTML

_Form is a way to collect the data from the users. In HTML we can create forms using `<form>`. It accepts multiple attributes as well such as action attribute which is responsible for where the data needs to be redirected after submission of the forms.Form accepts two HTTP methods only `get` and `post`._

## Inputs

_Input is a tag in HTML generally used to create input field where user can fill the data. There are multiple types of inputs in HTML by default it has text type._

## Types of inputs and attributes accepted

1. `text`: It holds the text data for users such as name, username or first name, last name etc.
2. `password`: This input is used to get the confidential data such as account password etc.
3. `email`: Generally used to collect the email data from a user. It is a self validated input.
4. `tel`: This input is used to collect the contact information from a user.
5. `number`: This input will be used to contain the numeric data such as age.
6. `radio`: This input creates a circular button or radio button where we can collect the data like genders. It allows you to select only one option among multiples
7. `checkbox`: This input creates checkboxes and generally used to collect the data like skills. Here we can able to select multiple options among multiple.
8. `file`: This input is used to collect the media files from users such as images videos audios etc.
9. `date`: This input allows you to collect the date data from the users here we can collect the data such as date of birth time etc.
10. `url`: This input helps you to collect the data like URL for example I want to collect profile URL, github URL etc.
