Selectors in HTML

  • simple Selectors
    1. ID uses symbol # to select the element
    2. Class uses symbol . to select the elements
    3. Tag name selector: using tag name we can select the elements
    4. Grouping Selector: uses , to group multiple elements at a time
    5. Universal Selector: selects all elements and uses * symbol
  • Combinator Selectors
    1. Descendent Selector
    2. Child Selector
    3. Adjacent Sibling Selector
    4. General Sibling Selectors
  • Attribute Selectors
  • Pseudo Class Selectors
  • Pseudo Element Selectors
